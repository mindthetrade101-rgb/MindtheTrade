## 📖 Overview

This repository contains the complete source code for the **MTT (MindTheTrade) Precision Trading Framework** website – a professional online presence for a proprietary trading methodology. The site is designed to:

- Present a **structured, multi‑timeframe trading approach** (MPAP/WPAP/DPAP/H4PAP/H1)
- Show **verified live performance** (Myfxbook widget)
- Offer **trading services** (insights, mentorship, capital program)
- Promote **Introducing Broker (IB) partnerships** with XM and IC Markets
- Build **trust and authority** through founder story, trade examples, and FAQ

**The website does NOT sell any EA or robot.** The execution engine is an internal tool; the public face is the MTT framework itself.

---

## 🛠️ Technology Stack

- **HTML5** – Semantic structure  
- **CSS3** – Custom styling, responsive grid/flexbox, dark theme  
- **JavaScript** – Mobile menu toggle, FAQ accordion, simple form handling  
- **Font Awesome 6** – Icons  
- **Google Fonts (Inter)** – Typography  

No build steps, no dependencies – pure static website.
